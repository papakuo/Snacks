    7. The Tower of Babel (Genesis 11: 4 - 7)

    The Starter

    People like to build the so-called tallest building. If it's not in the world then the country. If it's not the country then the state. If it's not the state then the city. If it's not a foreseeable building then it's the wealth, power, reputation, etc. In other words, something to make a name for ourselves. Because there is a Tower of Babel in everybody's mind. Is it really worthwhile? If it's not then what should we do?
